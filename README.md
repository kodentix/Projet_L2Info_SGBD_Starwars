# Projet Base de Données — Univers Star Wars  
Licence 2 Informatique — Projet Base de Données (2025)

---

## Description du Projet

Ce projet a été réalisé dans le cadre du cours de Bases de Données en L2 Informatique.  
L’objectif est de modéliser puis d’implémenter une base de données complète basée sur l’univers de Star Wars, incluant :

- Les films
- Les séries
- Les personnages
- Leurs relations d’apparition (dans films et séries)

---

## Travail Réalisé

### Étape 1 — Modélisation (E/A)

- Analyse des entités possible  
- Définition des entités retenus, attributs et cardinalités  
- Création du modèle E/A et justification complète  

[Voir Etape 1](Projet/1ere_Etape/Etape1_Modélisation.pdf)

Contenu :
- Entités : Films, Séries, Personnages
- Associations : Joue_film, Joue_serie
- Gestion des attributs spécifiques (chronologie BBY/ABY, durée, rôle, acteur)

---
